1. Сплошная стена из одинаковых брусьев:

	A. Выбор существующего типа стены в списке
		"Выберите тип стены:"
	B. Выбор загруженного семейства бруса
		из списка "Выберите профиль:"
	C. Выбор материала для бруса:
		"Выберите материал: ->
		  -> Выбор категории(правее надписи) ->
		  -> Выбор материала(правее выбора категории)"
	D. Указание рабочей высоты выбранного бруса:
		поле с надписью "Рабочая высота"
	E. Указание смещения бруса относительно оси стены:
		поле с надписью "Смещение"

*Как правило, для однослойной стены с осью в середине стены:
		"Смещение" = -(Толщина сечения бруса / 2)

	F. Указать начальную и конечную отметку:
		поля с надписями "Начать с отметки"
				   "До высоты" *по умолчанию 12000
	G. Нажатие кнопки "Замостить стену профилем"



2. Сплошная стена из одинаковых брусьев, где нижний брус полувенец:

*См пункт 1 с шага A до шага E, затем:

	F. Нажатие кнопки "Выбрать первый профиль"
	G. Выбор загруженного семейства бруса
		из списка "Выберите профиль:"
	H. Выбор материала для бруса:
		"Выберите материал: ->
		  -> Выбор категории(правее надписи) ->
		  -> Выбор материала(правее выбора категории)"
	I. Указать начальную отметку полувенца:
		поле с надписью "Начать с отметки"
	J. Указать рабочую высоту полувенца:
		поле с надписью "Рабочая высота"
	K. Нажатие кнопки "Установить первый профиль"

*В главном окне поле под надписью "Начать с отметки" автоматически
заполнится данными, которые суммируют прежде введенные высоты 
("Начать с отметки" и "Рабочая высота") из закрытого окна
	
	L. Указание рабочей высоты основного бруса:
		поле с надписью "Рабочая высота"
	M. Указание смещения бруса относительно оси стены:
		поле с надписью "Смещение"

*Как правило, для однослойной стены с осью в середине стены:
		"Смещение" = -(Толщина сечения основного бруса / 2)

	N. Указать конечную отметку:
		поле с надписью "До высоты"  *по умолчанию 12000
	O. Нажатие кнопки "Замостить стену профилем"



3. Работа с семействами бруса (Профиль/Sweep):

	А. Кнопка "Load" загружает семейство в проект из .rvt файла
	B. Кнопка "ReName" позволяет изменить название семейства профиля
	C. Кнопка "Delete" удаляет семейство профиля из проекта
	D. Кнопка "Edit" позволяет изменить семейство профиля, используя стандартный
		редактор семейства программной среды Revit
	E. Кнопка "Copy" копирует семейство профиля в проекте, добавляя "_Copy" к названию
